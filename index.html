Glimpse Privacy Policy Last updated: June 15, 2026 Provided by UWAGAPRIME LLC. Contact: uwagaprimellc@gmail.com This policy explains what the Glimpse browser extension does with data. Glimpse is a workplace tool that an employer installs on company devices. It helps a business see when sensitive information is being entered into public AI tools such as ChatGPT, Claude, Gemini, Copilot, and Perplexity. What data Glimpse handles When an employee uses one of the supported AI tools, the extension reads the text submitted in the prompt, and on ChatGPT it also reads the text of documents attached to the chat, so it can check for sensitive information. Glimpse does not read general web browsing, keystrokes outside these AI tools, passwords, or personal accounts. How the data is processed Before anything leaves the device, Glimpse masks common structured secrets such as Social Security numbers, card numbers, emails, and API keys, replacing them with placeholders like [ssn]. On supported machines, the analysis runs on the device itself using the browser's built-in AI, so the prompt is not sent anywhere. On devices that cannot run the on-device model, only the masked version of the text is sent to a backend service operated by UWAGAPRIME LLC to score the risk. In that case the masked text may be processed by an AI provider (Anthropic) acting as a service provider. Only prompts judged to carry real risk are recorded. Harmless prompts are not stored. What is recorded and where it goes When a prompt is flagged, Glimpse records an event containing the AI tool used, the risk level, the category of data (for example "client data"), the time, and a short masked sample of the text. These events are shown only on a password-protected dashboard available to the employer who deployed Glimpse. Records contain only masked samples, not the full prompt, and structured secrets are masked before anything is sent. Data sharing Glimpse does not sell user data, does not transfer it to third parties for purposes unrelated to the tool's single purpose, and does not use it for advertising or creditworthiness. The only third party involved is the AI provider used to score masked text on devices that fall back to the backend, acting solely as a processing service. Data retention Recorded events are kept for 90 days by default and then deleted automatically. An employer can request a shorter window, or a longer one where a specific compliance rule requires it, and can have events deleted on request at any time. Security Data travels over encrypted connections. Structured secrets are masked on the device before anything is sent, and stored records keep only a masked sample rather than the full prompt. Who controls the data The employer who installs Glimpse controls the data it produces. UWAGAPRIME LLC provides the software and hosting on the employer's behalf. Employees with questions should contact their employer first. Your choices Glimpse can be configured to run fully on the device, with no data sent off the machine, if the employer chooses that mode. Employees should refer to their employer's own policy for how the tool is used in their workplace. Changes to this policy This policy may be updated. The date at the top reflects the latest version.
